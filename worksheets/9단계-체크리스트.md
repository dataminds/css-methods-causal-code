# 아홉 단계 체크리스트 (낱장)

> 논문을 쓸 때 들고 가는 한 장. 각 단계에서 스스로 답해야 하는 물음과 그 답의 산출물(논문 어디에 남는지).
> 자율 시나리오(S4·S5)와 기말 과제의 제출 양식이기도 하다. 칸마다 한두 문장이면 된다. 상세 = 책 부록 D.

| 단계 | 스스로 묻기 | 산출물(논문의 자리) |
|---|---|---|
| 1 연구질문·가설 | 기술인가 인과인가? 가설에 변수·방향·비교가 다 있나? 자료 보기 전에 적었나? | 가설 목록(서론 끝·사전등록) |
| 2 설계·식별 | 인과를 어디서 사 오나? 그 설계가 사는 가정은? 이 효과를 잡을 검정력은? | 설계 진술 + 검정력 계산서(방법) |
| 3 자료·정제 | 행 = 무슨 단위? 제외 기준을 미리 정했나? 제외 후 n은? (종단) 이탈은 무작위인가? | 표본 흐름 + 균형·이탈 표(방법·결과 앞) |
| 4 측정 | 이 변수가 그 개념인가? 신뢰도는(알파·코더 간)? 실패 요소는 어떻게 처리했나? | 척도 서술 + 신뢰도 보고(방법) |
| 5 기술·시각화 | 분포를 봤나(치우침·극단값)? 표 1을 만들었나? | 표 1 + 필요시 분포 그림(결과 앞) |
| 6 가정 점검 | 이 분석이 성립하는 조건은? 지금 성립하나(균형·조작 점검·통제 논거·잔차)? | 점검 결과 문장 + 통제 논거(방법·결과) |
| 7 주 분석 | 가설 순서대로 했나? 계획한 분석 그대로인가(바꿨다면 표지)? | 핵심 네 종 세트(M·SD/검정/효과크기/CI)(결과) |
| 8 강건성 | 다르게 해도 사는가(공변인·제외·하위집단)? 위약·극단값 시험은? 안 나온 결과도 자리에 있나? | 강건성 문단 또는 부표(결과 뒤) |
| 9 해석·보고 | 설계가 산 만큼만 말했나(인과어 점검)? 크기를 정직하게? 한계는 실명·정량·해소 경로로? | 결론 + 한계 절 + 공개 자료 링크(논의) |
| + 검증 로그 | 분석마다 3종(독립 재계산·극단값·재실행)을 코드로 했나? | 검증 로그(부록 B 양식 ; 제출물 동봉) |

## 쓰는 법

- 논문 시작 전: 1·2를 먼저 채운다(자료 보기 전). 이 두 칸이 비어 있으면 아직 분석을 시작할 때가 아니다.
- 분석 중: 3~8을 순서대로. 건너뛴 칸이 곧 심사평이 된다.
- 투고 전: 9와 검증 로그로 마감. 특히 "인과어 점검" — 모든 동사를 훑어 설계가 지불하지 않은 인과어("높인다"·"효과")를 걷어 낸다.
